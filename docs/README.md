# Image Harvest — Public Documentation

> Image Harvest 的公开文档仅包含 Chrome Web Store 上架材料。产品规划、营销策略、推广内容等商业敏感文档存放于独立私有仓库（`image-harvest-private`），不随本开源仓库分发。

## 🔒 私有文档说明

以下内容**不包含**在本开源仓库中：

- **产品需求文档（PRD）** — 未来功能路线图
- **营销策略文档** — 平台分级、时间表、投放计划
- **推广内容模板** — 34 个中文平台 + 34 个英文平台的发帖文案
- **追踪表模板** — Notion / Sheets / 飞书公式

这些文档属于**商业运营资产**，保留在项目作者的私有仓库中。如果你是用户或潜在贡献者，无需访问这些文档即可理解和使用本项目。
